##  安装方法（宝宝版）

### 第一步：下载插件文件
点击页面右上角绿色按钮 `Code` → 选择 `Download ZIP`  
下载完成后**解压**到任意文件夹

### 第二步：打开 Chrome 扩展页面
在浏览器地址栏输入：
```
chrome://extensions/
```
按回车进入
（其他的浏览器也行，edge一样操作，换一下前面就行）
### 第三步：开启开发者模式
页面**右上角**找到「开发者模式」，打开开关 🔘

### 第四步：加载插件
点击左上角 **「加载已解压的扩展程序」**  
选择你刚才解压的文件夹  
✅ 看到插件出现在列表里就成功了！

##  使用方法

1. 打开小红书内部 AI 平台：`https://ai.devops.xiaohongshu.com/`
2. 插件自动运行，页面会出现**时间轴导航面板**
3. 点击时间轴节点可快速跳转到对应对话位置

---

## ❓ 常见问题

**Q：插件装好了但没有反应？**  
→ 确认当前网址是 `ai.devops.xiaohongshu.com`，插件只在该网站生效

**Q：更新插件后不生效？**  
→ 去 `chrome://extensions/` 页面点击插件的 🔄 刷新按钮

**Q：浏览器重启后插件消失了？**  
→ 重新回到 `chrome://extensions/` 检查插件是否被禁用
